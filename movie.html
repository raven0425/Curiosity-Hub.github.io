Step 1: Simple Search Page & API Reality Check

We begin with a minimal webpage containing:

  • A single search bar
  • A search button
  • No styling frameworks

Important clarification:
IMDb does not provide a free public API for developers. This means we cannot fetch movie data directly from IMDb.

To proceed legally and reliably, we use the OMDb API (Open Movie Database), which offers IMDb-backed data and a free API key for development.
